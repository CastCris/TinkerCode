## Description
This circuit simulate a dice.

The circuit display a arbitrary number, between 1-6, into a led matrix 2x3 in a vertical/horizontal orientation.

There are four push-button that change the number or the draw led orientation. The push-buttons actions, from left to right:

Generate a new random number;

Increase the number;

Decrease the number;

Change the draw orientation. If horizontal, change to vertical and vice-versa.
